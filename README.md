# Gas Price Analysis in France (Spark Mini Project)

## Project Overview
This project uses **PySpark** to process and analyze a dataset of over 14 million gas price records in France from 2022 to 2024.

## Key Features
- **Data Collection**: Automatic download and merging of multi-year datasets.
- **Data Preparation**: Normalization, feature engineering (lags, rolling averages).
- **Visualization**: Interactive maps (Folium) and statistical plots (Seaborn/Matplotlib).
- **Modeling**: Random Forest Regressor to forecast next-day prices.
- **EV Impact**: Analysis of how EV charging stations affect traditional fuel pricing.
